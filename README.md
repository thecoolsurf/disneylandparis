## Use

### Frontend
Use ReactJS to build frontend view for client.\
Build with Dockerfile.
Run with Docker Compose.

### Backend
Use NodeJS, MySQL and Express to manage datas.\
Build with Dockerfile.
Run with Docker Compose.
